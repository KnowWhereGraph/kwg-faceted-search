Welcome to the Expert System for Humanitarian Relief

Note: This platform is in BETA, if you have any feedback, please don't hesitate to get back to us.

This platform is a prototype of the Expert System designed to support the non-profit organization - Direct Relief - to manage experts in response to disasters. The ultimate goal is to efficiently find the right experts with the right expertises at the right place and at the right time. This prototype is built based on the network of experts who are working on the COVID-19 Mobility Data.

There are three components of this system. The graph query endpoint allows one to use SPARQL query language to retrive information from the built-in Expert Knowledge Graph. The similarity interface enables a user to compare experts based on their representative works. And the expert explore interface provides a search engine empowered by the principle of follow-your-nose search. Users can choose to start the search based on either Expertise Topic , Hazard , or Place.

This prototype is part of the KnowWhereGraph project, which is funded by the NSF Convergence Accelarator Program with award number: 2033521.
